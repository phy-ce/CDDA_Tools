# CDDA_Tools

A collection of utilities for Cataclysm: Dark Days Ahead (CDDA) and
Cataclysm: Bright Nights (CDBN).

## Tools

| Tool | Description |
|------|-------------|
| [**CDDA_Installer**](CDDA_Installer/README.md) | Windows GUI to install, launch, and mod-manage CDDA & CDBN versions ([한국어](CDDA_Installer/README.md) / [English](CDDA_Installer/README.en.md)) |
| [**CDDA_Recipes**](CDDA_Recipes/README.md) | Offline crafting helper (local web app) — "how do I make X / what can I make with X / what do I get disassembling X", read from your install's JSON. Clickable item links, crafting trees, UI + names in EN / 한국어 / 日本語. |

## Download

No Python required — grab the latest Windows exes from the
[**Releases**](../../releases) page (both are attached to the same release):

- **`CDDA-Manager.exe`** — the installer / version manager
- **`CDDA-Recipes.exe`** — the offline crafting helper

Or run either from source (Python 3.8+, standard library only). See each tool's
own README for usage details.
